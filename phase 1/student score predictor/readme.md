# 📊 Student Score Prediction (Linear Regression)

## 📌 Overview

This project predicts a student's exam score based on the number of hours studied using a **Linear Regression model**.

It is a beginner-level machine learning project focused on understanding the complete ML workflow.

---

## 🎯 Problem Statement

Can we predict how well a student will perform based on how many hours they study?

---

## 🧠 Concepts Used

* Train-Test Split
* Linear Regression
* Model Training
* Prediction
* Evaluation Metrics (MSE, R² Score)

---

## 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn

---

## 📂 Dataset

A simple dataset containing:

* **Hours Studied (Input feature)**
* **Score (Target variable)**

---

## ⚙️ Workflow

1. Data Preparation
2. Data Visualization (Scatter Plot)
3. Train-Test Split
4. Model Training using Linear Regression
5. Predictions on test data
6. Model Evaluation

---

## 📈 Results

* **Mean Squared Error (MSE):** 8.33
* **R² Score:** 0.99

👉 The model performs very well and fits the data closely.

---

## 📊 Visualization

* Scatter plot of actual data
* Regression line showing predictions
